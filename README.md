# Knight Board Game

체스의 **나이트(Knight)** 이동 방식을 활용한 웹 게임 프로젝트입니다.  
React + TypeScript + Vite를 사용하여 구현하고 있습니다.

## 프로젝트 목표

다음 순서대로 기능을 구현합니다.

1. 보드 그리기
2. 현재 말 위치 표시
3. 나이트가 갈 수 있는 칸 계산
4. 클릭하면 그 칸으로 이동
5. 규칙 추가

현재는 **1단계: 보드 그리기**까지 구현한 상태입니다.

## 기술 스택

- React
- TypeScript
- Vite
- CSS
